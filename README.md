\# Shopping Web App



A Java-based web application for browsing and managing a jewelry shopping experience.



\## Features

\- Browse jewelry products

\- Add to cart \& checkout

\- User registration \& login

\- Order tracking

\- Admin dashboard (manage products, orders, customers)

\- Support ticket system

\- Telegram bot integration



\## Tech Stack

\- Java (Servlets and JSP)

\- Maven

\- HTML / CSS / JavaScript

\- MySQL



\## How to Run

1\. Clone the repository

2\. Import into Eclipse or IntelliJ as a Maven project

3\. Configure your database in `DatabaseConnection.java`

4\. Deploy on Apache Tomcat

5\. Open `http://localhost:8080/shopping-webapp`
